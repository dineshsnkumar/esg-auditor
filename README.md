## ESG-AUDITOR

A Spring Boot Application that uses Retrieval Augmented Generation(RAG) to check whether the user submitted
sustainability report follows EU ESG guidelines along the factors of Environment, Social and Governance.

### Technical details

- Vector Store: PGVector:17
- LLM: Mistral
- Embedding Model: mxbai-embed-large

### Setup instructions

- As of January 2026, Spring Boot supports Spring AI dependencies between versions greater than 3.5.0 and less than
  4.0.0. Check to make sure Spring Boot AI is supported
