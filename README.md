## ESG-AUDITOR

A Spring Boot Application that uses Retrieval Augmented Generation(RAG) to check whether the user submitted
sustainability report follows EU ESG guidelines along the factors of Environment, Social and Governance.

### Technical details

- Vector Store: PGVector:17
- LLM: Mistral
